# cpp编程简介

## 本教程的两大部分

- Object Based：关注于单个class的设计，两种基本的class可以分为：
  - class without pointer members -- （Complex本教程的例子）
  - class with pointer members -- （String本教程的例子）
- Object Oriented：关注于class之间的关系
  - 继承（inheritance）
  - 复合（composition）
  - 委托（delegation）
